Claude Fable 5 の利用期間が延長された経緯

Anthropic の最上位モデル Claude Fable 5 を有料プラン(Pro / Max / Team / 一部 Enterprise)で追加料金なしに使えるプロモーションは、当初 2026年7月7日までの予定でしたが、まず 7月12日 23:59:59(太平洋時間)まで、続いて 7月19日 23:59:59(太平洋時間)まで 2 度延長されました。この短期プロモーション自体が、6月12日の米政府による輸出管理指令で Fable 5 が全世界で提供停止となり、7月1日に限られたキャパシティで再展開(redeploy)された「補償」として始まったもので、期間の短さへのユーザーの不満とキャパシティ確保の進み具合が延長の背景にあります。

経緯のタイムライン

前提: なぜ「期間限定」で提供されていたのか

Claude Fable 5 は 2026年6月9日に公開されましたが、わずか 3 日後の 6月12日、米政府の輸出管理指令(export control directive)に基づき、Claude Mythos 5 とあわせて全世界でアクセスが停止されました。報道では、モデルのジェイルブレイク(安全機構の回避)を巡る国家安全保障上の懸念が理由とされています。Anthropic は外部レッドチーミングなどの追加検証と対策(報告されたジェイルブレイクの再現試行の 99% 超をブロックする分類器の導入など)を経て、6月30日の規制解除を受け 7月1日にサービスを再開しました。

この再開(Anthropic は公式ブログ「Redeploying Claude Fable 5」で発表)にあわせて、停止期間の補償も兼ねた期間限定プロモーションが始まりました。内容は次のとおりです。

  • 対象: Pro / Max / Team プランおよびシートベースの Enterprise プラン(組織で有効化した場合)
  • 期間: 当初は 2026年7月7日まで
  • 上限: 週次利用上限の 50% までを Fable 5 に充当可能(追加割当ではなく、既存の週次上限の内数)
  • 利用面: Web / モバイル / デスクトップの Claude、Claude Code、Claude Cowork などで利用可能

再開時点ではデータセンターのキャパシティが限られていたため、「無制限・約 2 週間」ではなく「50% 上限・約 6 日間」という圧縮された条件になったと報じられています。

第 1 回延長: 7月7日 → 7月12日

当初期限の 7月7日当日(米国時間の深夜)、Anthropic は期限を 2026年7月12日 23:59:59(太平洋時間、日本時間では 7月13日 15:59:59)まで延長すると発表しました。提供期間がわずか 6 日程度と短く、期限直前に駆け込み利用していたユーザーから不満の声が上がっていたことが延長の背景として報じられています(Android Authority は「早すぎる打ち切りへの反発を受けての延長」と報道。Anthropic が公式に延長理由を明言したかは未確認)。

第 2 回延長: 7月12日 → 7月19日

期限を目前にした 7月11日頃、Anthropic は再度の延長を発表し、期限は 2026年7月19日 23:59:59(太平洋時間)になりました。あわせて、Claude Code の週次利用上限を 50% 引き上げる措置も同日まで延長されています。Anthropic は「キャパシティが確保でき次第、Fable 5 を有料プランの標準提供に戻したい」との意向を示していますが、その時期は明示していません(標準提供へ戻す時期は未確認)。

プロモーション終了後はどうなるか

期限(現時点では 7月19日 PT)を過ぎると、サブスクリプションに含まれる形での Fable 5 利用は終了し、継続利用には次のいずれかが必要になります。

  • Usage Credits(利用クレジット)を購入してアプリ内で利用する
  • API 経由で従量課金で利用する(入力 $10 / 出力 $50 per 1M トークン)

まとめ

「利用期間延長」は単発のキャンペーン延長ではなく、(1) 輸出管理指令による 19 日間の全面停止、(2) 限られたキャパシティでの再展開と補償的な短期プロモーション、(3) 期間の短さへの反発とキャパシティ確保を受けた 2 度の期限延長、という一連の流れの結果です。最新の期限や条件は Anthropic の公式発表(Redeploying Claude Fable 5 ページおよび Help Center)で確認することをおすすめします。
